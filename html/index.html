Arduino Library for the Notecard

Introduction

The note-arduino Arduino library for communicating with the Blues Wireless Notecard via serial or I2C.

This library allows you to control a Notecard by writing an Arduino sketch in C or C++. Your sketch may programmatically configure Notecard and send Notes to Notehub.io.

Dependencies

This library is a wrapper around and depends upon the note-c library, which it includes as a git submodule.

In addition, this library requires a physical connection to a Notecard over I2C or Serial to be functional.
